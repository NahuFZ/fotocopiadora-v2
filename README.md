1. Para armar la estructura de la base de datos en MySQL, hay un archivo "fotocopiadora.sql" dentro de la carpeta "fotocopiadora-v2/sql". Así que lo deben importar.
2. Para establecer la primera cuenta de administrador, hay un archivo "setup_test.sql" en la misma carpeta. También hay que importarla. 
3. El usuario es "admin@fotocopiadora.com" y la contraseña "admin123".
4. Idealmente, deberían abrir el archivo "login.jsp" para iniciar sesión pero igualmente cualquier archivo JSP los redirige al login si no tienen una sesión abierta.
5. La carpeta donde se guardan los archivos subidos es C:\Users\{user}\archivos-fotocopiadora
6. Por las dudas aclaro, funciona con Tomcat v11.0 y J-Connector v9.3.0
